# 🖥️ TechStore - Tienda de Componentes de Computadoras

## 📋 Presentación del Proyecto

**TechStore** es una plataforma de comercio electrónico para la venta de componentes y periféricos de computadoras.

Este proyecto universitario simula una tienda online real aplicando tecnologías modernas de desarrollo full-stack y buenas prácticas de arquitectura de software.

El sistema permite:

- Navegar por un catálogo de productos
- Filtrar por categorías
- Agregar productos al carrito
- Completar el proceso de compra
- Gestionar productos y órdenes desde un panel administrativo

---

## 🎯 Objetivos del Proyecto

### Objetivos Generales

- Desarrollar una aplicación web full-stack funcional
- Implementar un sistema completo de e-commerce
- Aplicar buenas prácticas de arquitectura y desarrollo
- Demostrar competencias en frontend, backend y bases de datos

### Objetivos Específicos

- **Frontend:** Interfaz responsiva y accesible
- **Backend:** API RESTful segura
- **Base de Datos:** Modelo relacional eficiente

- **Testing:** Pruebas unitarias e integración
- **DevOps:** Contenedorización con Docker

---

## 🚀 Tecnologías Utilizadas

### Frontend

- **Next.js 14/15 (App Router):** Framework de React con SSR y SSG optimizado para rendimiento y SEO.
- **React 18+:** Biblioteca para construir interfaces basadas en componentes.
- **TypeScript:** Tipado estático para mayor seguridad y mantenibilidad.
- **Tailwind CSS:** Framework utility-first para diseño rápido y responsivo.
- **shadcn/ui:** Componentes UI accesibles y personalizables.
- **Zustand:** Gestión de estado global ligera (carrito y autenticación).

### Backend

- **Node.js:** Entorno de ejecución para JavaScript del lado del servidor.
- **Express.js:** Framework para construir la API RESTful.
- **TypeScript:** Consistencia de tipos entre frontend y backend.

### Base de Datos

- **PostgreSQL:** Base de datos relacional robusta y escalable.
- **Prisma ORM:** ORM moderno con tipado seguro y sistema de migraciones.

### Autenticación y Seguridad

- **JWT:** Autenticación basada en tokens.
- **bcrypt:** Hash seguro de contraseñas.
- Validación de datos en frontend y backend
- Protección contra inyección SQL mediante Prisma
- Variables de entorno para datos sensibles
- Configuración de CORS y rate limiting

### Testing

- **Vitest:** Pruebas unitarias rápidas.
- **React Testing Library:** Testing de componentes.
- **Supertest:** Pruebas de endpoints HTTP.

### Servicios Adicionales

- **Multer:** Carga de imágenes de productos.
- **Nodemailer:** Envío de correos (modo local).
- **Stripe (modo test):** Simulación de pagos.
